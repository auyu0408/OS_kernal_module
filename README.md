# OS2021_HW1
## Setup  
1. Install hooks and build app
```bash=
make
```
2. Build and install module
```bash=
#in module/
make
make ins
```
3. Remove module
```bash=
#in module/
make rm
```
4. Delete other file except source code in module
```bash=
#in module/
make clean
```
5. Remove app
```bash=
make clean
```
## Useage
- execute app
```bash=
./app
```
